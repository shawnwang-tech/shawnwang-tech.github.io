PyTorch 可视化工具及调试利器 TensorboardX

TensorboardX 与 visdom 的区别

  • TensorboardX 可以把多组实验数据横向比较，visdom 不行。visdom 更像是一个 web 版本的 matplotlib，缺乏像 TensorboardX 一样更高层的封装。例如想要在 visdom 中显示 loss 曲线，你可能要维护一个所有 loss 值的 list （需要自己写一些辅助代码），每次显示整个 list。而 TensorboardX 中要做的只是 writer.add_scalar
  • TensorboardX 可以看到网络结构，visdom 不行。对于深度学习初学者来说，简直不要更爽，神经网络结构如庖丁解牛一般呈现在你的面前。

注意

  • 写 loss 的时候，要加上横轴的值，否则会出现不同的 epoch 叠加在一起的现象。data-overlapped。需要借助一个 tranloss_list 全局变量实现。然后就可以得到完整的数据了
